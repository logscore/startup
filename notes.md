## CSS 

- To make a button into an image or svg, do this: 
```html
    <button>
        <img src="/path/to/image" />Words showup if you put them here
    </button>
```

--OR--

```html
    <style>
        .img_button{
            background-image: url(/path/to/image)
        }
    </style>

    <button class="img-button"></button>
```
